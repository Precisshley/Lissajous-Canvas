Getting Started

Use the inputs on the left to adjust the parameters of the Lissajous curves:

  • Distance: Adjusts the distance of the Lissajous curve.
  • Transparency: Adjusts the transparency of the background, lines, or fill.
  • Radians: Sets the angle in radians.
  • Polarity: Adjusts the polarity of the curve.
  • X: Determines the trigonometric function and multiplier used for the X-axis.
  • Y: Determines the trigonometric function and multiplier used for the Y-axis.
  • Lines: Controls the connection style or spacing of the lines.
  • Amount: Adjusts the number of curves or repetitions.
  • Twist: Alters the twist effect applied to the curve.
  • BlendMode: Sets the blending mode for the curve rendering.
  • Colour: Allows you to select colors for the background and outlines.
  • Multiplier x-axis: Chooses a multiplier for the X-axis values.
  • Multiplier y-axis: Chooses a multiplier for the Y-axis values.

For more detailed explanations, refer to the documentation or help section.
